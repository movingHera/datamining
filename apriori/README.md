##Apriori算法

    Apriori(filename, min_support, item_start, item_end)

####参数说明

* filename:(路径)文件名
* min_support:最小支持度
* item_start:item起始位置
* item_end:item结束位置

####Example

```python
import apriori
c = apriori.Apriori('basket.txt', 11, 3, 13)
```
